Ecoinvent tool (user interface draft)

This is a first placeholder to start drafting the interface that will request the user's inputs (not yet a draft for the design, this goes last). The goal is to have a very clear menu structure for the inputs before moving forward to create the backend (models). The inputs are:

  • all numeric fields to be requested from the user
  • all units from the numeric fields
  • all yes/no questions
  • all dropdown menus
