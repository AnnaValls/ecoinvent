Ecoinvent tool (user interface draft)

This is a first draft to start discussing the interface that will request the user's inputs (not yet a draft for the design). The goal is to have a very clear menu structure for the inputs before moving forward to create the backend (models). The inputs should include:

  • all numeric fields
  • all units
  • all default values
  • all possible options for dropdown menus.
Please send comments, requests or any other feedback to:
  • lbosch@icra.cat (Lluís Bosch)
  • lcorominas@icra.cat (Lluís Corominas)
Optionally, a word document with the same tree structure as this web page could be useful as a reference to code this page.

  3. Treatment plant characteristics
    1. Liquid treatment line unit processes
      1. Pumping
      2. Preliminary: to remove trash and grit
        1. Screening
        2. Degritting
        3. Neutralization
        4. Equalization
      3. Primary: to remove settleable matter
        1. Primary clarification
        2. Chemical addition
      4. Secondary: to remove organic matter (BOD5, TSS)
        1. Activated sludge
        2. Lagoon (anaerobic, facultative, aerobic)
        3. Constructed wetlands
      5. Tertiary: to remove ammonia, total nitrogen, total phosphorus, metals
        1. Multistage activated sludge
        2. Multistage biofilm processes
        3. Chemical addition
      6. Advanced: to reach low levels of TSS, BOD5, total nitrogen, total phosphorus, pathogens
        1. Filtration
        2. Ultrafiltration
        3. Disinfection(e.g.Cl2,UV,O3)
        4. Activated carbon
    2. Solids treatment line unit processes
      1. Thickening: to increase the solids content
        1. Gravity thickening
      2. Digestion: to stabilize organic matter
        1. Anaerobic digestion
        2. Aerobic digestion
      3. Conditioning: to improve dewatering
        1. Coagulant and polymer addition
      4. Dewatering: to increase the solids content
        1. Centrifugation
        2. Filter pressing
        3. Vacuuming
      5. Nutrient treatment and recovery
        1. Struvite formation
        2. Anammox
      6. Drying: to increase the solids content
        1. Sludge beds
        2. Heat drying
      7. Sludge or biosolids disposal
        1. Composting
        2. Land application
        3. Landfilling
        4. Incineration
