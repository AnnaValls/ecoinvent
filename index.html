Ecoinvent web tool

This is a placeholder for drafting the interface that will request the user's inputs (not yet a draft for the design, this goes last). The goal is to have a very clear user interface for the inputs before moving forward to create the backend (models). The inputs are:

  • all numeric fields to be requested from the user
  • all units from the numeric fields
  • all yes/no questions
  • all dropdown menus
